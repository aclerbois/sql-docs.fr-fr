# <a name="generalflags-property-sinstance-class"></a>Propriété GeneralFlags (classe SInstance)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  Obtient les indicateurs généraux associés à l’instance de [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
object.GeneralFlags [= value]  
```  
  
## <a name="parts"></a>Éléments  
 *objet*  
 Un [classe SInstance](../../../relational-databases/wmi-provider-configuration-classes/sinstance-class/sinstance-class.md) objet qui représente une instance de serveur.  
  
## <a name="property-valuereturn-value"></a>Valeur de propriété/valeur de retour  
 Tableau des objets qui représentent les indicateurs généraux pris en charge par l'instance de [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].
