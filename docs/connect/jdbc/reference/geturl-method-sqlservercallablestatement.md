# <a name="geturl-method-sqlservercallablestatement"></a>Méthode getURL (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère la valeur du paramètre désigné en tant qu’objet URL dans le langage de programmation Java.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom| Description|  
|----------|-----------------|  
|[getURL (int)](../../../connect/jdbc/reference/geturl-method-int.md)|Récupère la valeur du paramètre désigné en tant qu’objet dans le langage de programmation Java fonction de l’index de paramètre URL.|  
|[getURL (java.lang.String)](../../../connect/jdbc/reference/geturl-method-java-lang-string.md)|Récupère la valeur du paramètre désigné en tant qu’objet URL dans le langage de programmation Java selon le nom de paramètre.|
