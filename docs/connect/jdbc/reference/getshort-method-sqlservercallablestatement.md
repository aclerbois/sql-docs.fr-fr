# <a name="getshort-method-sqlservercallablestatement"></a>Méthode getShort (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Récupère la valeur du paramètre désigné en tant qu’un **court** dans le langage de programmation Java.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom| Description|  
|----------|-----------------|  
|[getShort (int)](../../../connect/jdbc/reference/getshort-method-int.md)|Récupère la valeur du paramètre désigné en tant qu’un **court** en fonction de l’index de paramètre de langage de programmation Java.|  
|[getShort (java.lang.String)](../../../connect/jdbc/reference/getshort-method-java-lang-string.md)|Récupère la valeur du paramètre désigné en tant qu’un **court** en fonction du nom du paramètre de langage de programmation Java.|
