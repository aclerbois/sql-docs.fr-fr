# <a name="length-method-sqlserverblob"></a>Méthode length (SQLServerBlob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Retourne le nombre d'octets dans l'objet BLOB.  
  
## <a name="syntax"></a>Syntaxe  
  
```  
  
public long length()  
```  
  
## <a name="return-value"></a>Valeur retournée  
 A **long** valeur qui spécifie le nombre d’octets.  
  
## <a name="exceptions"></a>Exceptions  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Notes  
 Cette méthode de longueur est spécifiée par la méthode de longueur dans l’interface java.sql.Blob.
