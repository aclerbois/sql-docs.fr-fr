# <a name="setclob-method-sqlserverpreparedstatement"></a>Méthode setClob (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Définit le paramètre désigné à un objet java.sql.Clob.  
  
## <a name="overload-list"></a>Liste de surcharge  
  
|Nom| Description|  
|----------|-----------------|  
|[Méthode setClob &#40;int, java.io.Reader&#41;](../../../connect/jdbc/reference/setclob-method-int-java-io-reader.md)|Définit le paramètre désigné à un objet java.sql.Clob.|  
|[Méthode setClob &#40;int, java.sql.Clob&#41;](../../../connect/jdbc/reference/setclob-method-int-java-sql-clob.md)|Définit le paramètre désigné à l’objet donné java.sql.Clob.|
