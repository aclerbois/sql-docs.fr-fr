# <a name="programming-with-ado-md"></a>Programmation avec ADO MD
Pour utiliser ADO MD avec votre outil de développement, vous devez établir une référence à la bibliothèque de types ADO MD. La description de la bibliothèque ADO MD est Microsoft ActiveX Data Objects (Multi-dimensional) Library. Le nom du fichier bibliothèque ADO MD est msadomd.dll, et l’ID de programme (ProgID) est « ADOMD ». Pour plus d’informations sur la façon de créer des références aux bibliothèques, consultez la documentation de votre outil de développement.
