# <a name="error-messages-odbc-driver-for-oracle"></a>Messages d’erreur (le pilote ODBC pour Oracle)
> [!IMPORTANT]  
>  Cette fonctionnalité sera supprimée dans une future version de Windows. Évitez d'utiliser cette fonctionnalité dans de nouveaux travaux de développement, et prévoyez de modifier les applications qui utilisent actuellement cette fonctionnalité. Au lieu de cela, utilisez le pilote ODBC fourni par Oracle.  
  
 Lorsqu’une erreur se produit, le pilote Microsoft ODBC pour Oracle retourne la valeur SQLSTATE (un code d’erreur ODBC) et un message d’erreur. Le pilote dérive ces informations d’erreurs détectées par le pilote et d’erreurs retournées par le serveur Oracle.
