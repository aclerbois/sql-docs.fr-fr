# <a name="visual-studio-analyzer"></a>Visual Studio Analyzer
> [!IMPORTANT]  
>  Prise en charge de Visual Studio Analyzer a été supprimée à compter de Windows 8 (Visual Studio Analyzer a été inclus uniquement dans les versions antérieures de Visual Studio.). Pour un autre mécanisme de résolution des problèmes, utilisez le suivi de l’offre.  
  
 Microsoft® Visual Studio™ Analyzer offre une vue d’ensemble de l’application ODBC. Le développeur peut évaluer, analyser et déboguer la structure, les performances et les interactions du point de vue de l’application, plutôt qu’à partir d’un composant ou le code du point de vue. Les informations collectées par Visual Studio Analyzer sont sous la forme d’événements, qui représentent une sorte de l’interaction entre les deux composants de l’application. Les événements peuvent être composées d’appels de fonction et retourne à partir de fonctions, telles que les connexions, les requêtes de base de données et les transactions.
